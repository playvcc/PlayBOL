Ladders Overview

Sector 7 uses XP ladders for seasonal competition. Teams earn XP by playing MLG‑ruleset matches.

  • 4v4 MLG BO2 – Main competitive ladder
  • 2v2 BO2 – Duo ladder
  • Sniper Ladder – Sniper‑only rules
  • Casual Ladder – Low‑stakes XP for casual squads

XP Rules (Per Match)

  • Win: +50 XP
  • Loss: +10 XP
  • Forfeit Win: +25 XP
  • Forfeit Loss / No‑Show: 0 XP

XP values can be tuned later in Supabase.
